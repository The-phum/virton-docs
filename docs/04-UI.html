4. UI 구성 요소#

버트온 웹 인터페이스는 모두 5가지 탭으로 구성되어 있습니다.

  • 대시보드 : 호스트 시스템의 실시간 현황 및 상태 확인

  • 인스턴스 : 각 노드의 하드웨어 정보 확인 및 개별 인스턴스 관리에 필요한 리소스 정보 제공

  • 리소스 : 스토리지 사용률, 연결 상태 및 CPU와 메모리 사용률 등 확인

  • 알림 : 클러스터, VM, 스토리지, 백업 등 발생하는 주요 이벤트 출력 및 로그 저장

  • 설정 : Proxmox 사용자 계정을 사용하여 API 연결 설정



4.1. 대시보드#


클러스터 전체 및 각 노드의 CPU, 메모리, 디스크, 네트워크 등 리소스 사용 현황을 통합하여 조회합니다. 또한 노드의 연결 상태, 업타임, 버전 서비스 상태 등을 실시간으로 모니터링하며, 관련 알림 정보를 확인할 수 있습니다.

연결한 Proxmox 클러스터 전체 현황과 개별 노드의 상태를 한 화면에서 통합적으로 확인할 수 있는 기능입니다.
대시보드는 CPU•메모리•디스크•네트워크와 같은 주요 시스템 리소스의 사용량을 시각화하여 제공하며, 각 노드가 현재 어떤 부하 상태에 있는지 직관적으로 파악할 수 있도록 구성되어 있습니다.

또한, 각 노드의 연결 상태(Online/Offline), 업타임(Uptime), 설치된 Proxmox VE 버전 및 주요 서비스 활성화 여부 등 운영에 필요한 핵심 정보를 실시간으로 모니터링할 수 있습니다.
이벤트가 발생한 경우, 관련 알림이나 로그를 통해 즉시 원인을 파악하고 대응할 수 있어 전체 클러스터의 안정적인 운영을 지원합니다.
